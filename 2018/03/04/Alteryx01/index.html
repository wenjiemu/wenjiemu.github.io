Alteryx系列之一：初来乍到

发表于 2018-03-04

1. 主要产品

Alteryx, Inc.总部设在加利福尼亚州的尔湾，其主要产品为：

  • Alteryx Designer——用于数据准备，混合和分析，可以部署在云端和内部；
  • Alteryx Server——一种安全可扩展的产品，用于在基于Web的环境中共享和运行分析应用程序；
  • Alteryx Analytics Gallery——基于云的协作产品，允许用户在中央存储库中共享工作流。

2. 发展历程

Alteryx公司为大约50个国家的商业和金融服务企业、消费品、医疗保健、零售、技术、旅游和酒店业客户服务。从Alteryx发展历程，我们可以知晓为什么空间分析是其特色，可以看到近几年资本市场对其数据处理能力的青睐。

  • 1997年，Alteryx的前身SRC LLC由Dean Stoecker, Olivia Duane Adams 和 Ned Harding创立；
  • 1998年，SRC 发布 Allocate，一款数据引擎，包含地理位置优越的美国人口普查数据，允许用户操纵，分析和映射数据；
  • 1998年，SRC 开发了Solocast；
  • 2000年，SRC LLC与美国人口普查局（U.S. Census Bureau）签订合约；
  • 2006年，SRC LLC发布Alteryx软件，一款统一的空间和非空间数据环境，用于构建分析过程和应用程序；
  • 2010年，SRC LLC更名，取其核心产品Alteryx之名，改为Alteryx, Inc.；
  • 2011年，Alteryx从SAP AG的投资部门SAP Ventures筹集了600万美元的风险投资；
  • 2013年，Alteryx 从 SAP Ventures 和 Toba Capital 获得了1200万美元的A轮投资；
  • 2014年，公司从Insight Venture Partners、SAP Ventures 和 Toba Capital获得了共计6000万美元的B轮投资；
  • 2015年，Alteryx 在C轮融资中获得了Iconiq Capital领投，Insight Venture Partners 和 Meritech Capital Partners跟投的8500万美元；
  • 2015年，Alteryx宣布与微软建立新的关系，以便通过Power BI实现更快，更轻松的数据分析；
  • 2016年，Alteryx位列Forbes Cloud 100第24名；
  • 2017年3月24日，Alteryx IPO登陆纽交所。

Alteryx是一家从20世纪走入21世纪的科技公司，90年代末是个神奇的时期，宛如科技寒武纪，爆发了很多知名公司。回看华夏大地，1998、1999、2000三年，腾讯、阿里、百度三家公司相继创立。20年后的这个时间段是否也能爆发一波下一周期的知名公司，让我们过十几年再来评说。

3. Gartner魔力象限

近几年，Alteryx在Gartner魔力象限的表现颇佳，2017年从此前的visionaries跃升为challengers，较2016年相比唯有三个厂家晋升，Alteryx是其中之一，另外两个为IBM和RapidMiner。当年的评论为：“提供了便于使用的数据科学平台，带有自主数据准备和高级分析的功能。除此之外，还具备了数据模拟和优化的能力。由于其稳定的客户增长，它从去年的有远见者象限跳跃上升至今年的挑战者象限”。2017年的新星还有H2O.ai和Dataiku。

2018年，Alteryx已经跃升为Leaders，今年H2O.ai急速飙升最令人瞩目。此外，领导者KNIME，RapidMiner，远见者Databricks，Dataiku以及利基者Anaconda都值得关注。初步了解，H2O.ai综合了Spark和mlib的优点，加之性能有优势，易上手。H2O.ai提供了一站式自动建模平台DAI（Driverless AI）DAI平台集成了三部分内容：

  • 1、jupter notebook：适用于对建模熟悉，编程熟悉的人；
  • 2、H2Oflow：适用于熟悉建模的人，快速数据分析；
  • 3、Driverless：适用自动建模。

顺便提一句，自动化的机器学习平台DataRobot也值得关注，这家公司在2017年还挖了Alteryx高管。

4. Alteryx的特点

Alteryx Designer采用直观的拖放界面(intuitive drag-and-drop)，可以创建工作流、宏和应用，并发布到Alteryx Analytics gallery，其他人可以下载运行。 Alteryx Designer处理来自Alteryx Engine本地实例的工作流程。而且通过Alteryx Server，工作流可以定时运行。Alteryx可以安装R工具套件。Alteryx引擎还可以发电子邮件，从网络上传下载数据。

The ETL ( Extract, Transform, and Load ) blueprint that Alteryx provides is superior to tools like Excel, Access, and SQL.

Moreover, predictive analytic models built within Alteryx can be quickly expressed with visualization tools such as Tableau, Power BI, and QlikView.

总体来讲，Alteryx有几大特色：

  • 拖放界面创建工作流，宏，应用，可复用，可定时运行；
  • 可发布云平台多人协作，集成了R套件，对接主流可视化软件；
  • 百万行级数据处理利器，方便多源数据融合。

6.本系列Blog的提纲

本系列将聚焦Alteryx的使用及设计思想，计划涵盖以下几方面：

  • 数据备和数据融合(Data preparation and blending)——the most powerful capabilities of Alteryx
  • 基本功能——公式计算、转换数据(Transforming Data)、数据解析(Data Parsing)
  • 高级功能——创建宏、云、自动调度、分析治理
  • 最佳实践——构建高效工作流，数据探索，处理数据源

通过梳理Altery的以上几方面内容，不仅可以熟悉该软件，还可以帮助理解该类软件的设计思想，积累类似工具的使用技巧及注意事项，避免走弯路。

7.Resource optimization and speed processing

开始之前，先列举一些工作流优化的注意事项。 Temporary Directory: It is best to have the temporary directory point to a physical hard drive separate from your boot drive. The Select, Filter, and Auto Field tools can be utilized to process records downstream in your workflows at a faster rate：

  • Removing fields that won’t be utilized downstream for the output.
  • Use the Filter tool to remove needless rows
  • Avoid string fields with large sizes（不必要的长度）

7.1性能分析

输出日志可以查看各工具的运行时间，该输出功能默认关闭。可以确定哪种工具需要处理的时间最长，以及比较在两台不同计算机上运行同一工作流程的差异。The Performance Profiling can be found under Workflow Configuration | Runtime | Enable Performance Profiling.

7.2连接数据
