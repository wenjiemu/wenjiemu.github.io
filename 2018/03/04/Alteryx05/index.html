Alteryx系列之五：最佳实践总结

发表于 2018-03-04

工作流设计的重要性：The data necessary for making critical business decisions is best consumed by streamlined workflows. 工作流布局的重要性：The workflow layout can help you visually build workflows faster.

工作流布局

  • 画布方向： select the Canvas tab and select Layout Direction
  • The Annotation option under the Canvas tab includes the Hide, Show, and Show w/ Tool Names.
  • The Connection Progress will have three options; Hide, Show, or Show Only When Running.

自动保存

  • The Autosave option is located under Options | User Settings | Edit User Settings | Advanced:
  • The Autosaved files can be accessed by going to File | Open Autosaved files. The available files are listed and can be opened or saved as a different file.

搂搂数

浏览工具包含数据分析信息，提供有关数据质量的详细信息，由以下颜色编码：

  • 红色：不好（可能有空白处）
  • 黄色：由于没有数据，为空
  • 灰色：空
  • 绿色：良好，数据质量是健康的

小技巧

  • 当Alteryx已经打开时，资源管理器框是快速打开工作流的好方法。工作流文件可以从Windows资源管理器中拖出并放到画布上。
  • 假设有很多列，并且浏览浏览工具查找列可能不太容易。在结果 - 浏览配置中，可以选择可用字段旁边（×和√旁边）的下拉区域。在工具配置中选择要查看的字段，并避免需要添加新工具（如选择工具）以缩小字段数量; 因此它也优化了性能。
  • 用容器组织工具，此前提到了
  • 对齐：Align Horizontally or Align Vertically. The shortcut favorites for this are Ctrl + Shift + - to align horizontally and Ctrl + Shift + + to align vertically.
  • 有时候通过输入数据工具引入了新字段，旧字段已被删除，无论是使用本地文件还是从数据库中删除。这些字段在工具中将显示为缺失，并且不需要保留用于下游处理。使用Options选项卡 下的工具配置中的“ Forget All Missing Fields”选项可以快速解决此问题。
