发表于 2018-03-04

1. 公式计算

1.1 Formula

  • start with N：StartsWith([LocationDesc], ‘N’)
  • returns the largest integer less than or equal to X：FLOOR(X)
  • return the smallest integer greater than or equal to the defined value： CEIL function
  • ROUND(SQRT(tonumber([DataValue])),0.01)
  • GetWord：The GetWord function will return the Nth word in the string. 注意，The first word starts at 0
  • PadLeft和PadRight：将指定的字符填充到字符串的左右侧
  • DateTimeNow()将提供工作流程的运行日期和时间，这对于在输出数据时创建更新的日期和时间非常有用，并且可以快速查看数据上次更新的日期和时间。该DateTimeToday()功能类似于它只提供日期，并删除时间组件。

《Learning Alteryx》在该章节使用了U.S. Chronic Disease Indicators (CDI)数据集,随书附赠的下载链接在谷歌云，科学上网下载速度太慢，可以去美国疾病防控中心官网下载，凌晨下载速度可达到4-10Mbps。

1.3 Multi-Field Formula

Create a new field that provides a new Record ID for every YearEnd field.会比较常用。

  • Select YearEnd from the Group By (Optional) options.
  • [Row-1:RecordID]+1
  • More than one group can be identified by selecting more than one Group By option.

2. 转换数据

在summarize部分强调用了Group By动作。The granularity of data can be defined within the Group By actions

2.1 转数据

Transpose and Crosstab tools

  • Transpose：矮胖变高瘦，好处为A calculation can be referenced on one field, rather than an individual calculation referencing each field if they were in their own columns.
  • Crosstab：与Transpose相反

两种方式都为了方便分析，根据需求选择

2.2 Summarizing and aggregating data

  • using group by functions
  • summing, math, min/max, string concatenation, grouping

有趣的例子

  • Find the longest and shortest question for each Topic
  • Find the distinct StratisfactionCategoryID1 values for each Topic

2.3 Running total

  • obtaining a cumulative sum per record
  • Running Totals are useful for getting a total for a group——马克group
  • a running total of DataValueAlt by YearEnd and Topic. The cumulative sum will be continuing to aggregate until a new YearEnd and Topic appears in the data set
